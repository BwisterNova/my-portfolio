# Bright Ayomide's Portfolio

Welcome to my personal portfolio project! My name is **Bright Ayomide**. This site showcases my work, skills, and experience as a web developer and designer.

## About This Project

This portfolio is built with **React** and **Vite**, using modern web technologies and best practices. It features:

- A responsive, accessible design
- Dark and light mode with a custom theme toggle
- Smooth navigation and mobile-friendly layout
- Contact form powered by Formspree
- Animated transitions and interactive UI
- Projects, skills, and experience sections

All code, design, and content are created and maintained by me.

## Technologies Used

- React 19
- Vite
- CSS Modules & CSS Variables
- Framer Motion, React Icons, Typewriter Effect
- Formspree (for contact form)
